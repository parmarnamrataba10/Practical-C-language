1. Celsius to Fahrenheit Converter

Converts temperature from Celsius to Fahrenheit.

Formula:

Fahrenheit = (9.0 / 5.0) * Celsius + 32



2.Gross Salary Calculator

Calculates gross salary using base salary and allowances.

Inputs:

Base Salary
HRA (House Rent Allowance)
DA (Dearness Allowance)
TA (Travel Allowance)

Formula:

Gross Salary = Base Salary * (1 + (HRA + DA + TA) / 100)


3.Triangle Third Angle Finder

Calculates the third angle of a triangle when two angles are provided.

Formula:

Third Angle = 180 - (First Angle + Second Angle)
